# high-low-boxd
Higher/lower rating game of movies using Letterboxd's film ratings to test film and letterboxd knowledge!

Given two movies: movie1 and movie2 and movie1's rating, guess if movie2's rating on letterboxd.com is higher or lower. 

The objective is to get as many comparisons correct in a row as possible.

Download the latest release and run high-low-boxd.exe to play the game!
